# Andrea Khan – Portfolio Website

Welcome to my personal portfolio repository!

I am a **senior frontend software engineer based in New York**, with **3+ years of professional experience building applications in React**. This site is a living showcase of my work, experience, and experiments in modern frontend development.

### Tech Stack (so far)

- **Framework**: React (with TypeScript)
- **Build tooling**: Vite
- **UI library**: MUI (Material UI) with Emotion for styling
- **Routing**: React Router
- **Language**: TypeScript
- **Tooling**: ESLint, TypeScript, Vite dev server

### Getting Started

- **Install dependencies**

```bash
npm install
```

- **Run the dev server**

```bash
npm run dev
```

Then open the URL printed in your terminal (typically `http://localhost:5173`) in your browser.
